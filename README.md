# Minecraft.Win10_Launcher
一键下载安装(管理)所有已发布的Minecraft For Windows10版,支持启动  
在切换版本的同时会自动备份数据
# 与原版本的区别  
1.重新设计了UI,在美化界面的同时,添加了一些载入过程的可视化,让用户清楚当前的处理状态  
2.添加了对未处理错误的捕捉,大幅度减少程序异常退出几率  
3.对原作者代码进行了一些修复,优化了处理逻辑,添加了备用部署方案  
4.添加自定义设置项(实验)  
5.添加中文支持
